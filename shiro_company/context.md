# 누나(보스) 컨텍스트

## 프로필
- 기계공학 전공
- 편의점 야간 근무 3년차 (안산)
- 출퇴근 버스 편도 1시간
- 쉬는 날은 평일, 낮에 잠

## 전문 분야
- AI 시뮬레이션, 프롬프트 엔지니어링
- NovelAI 이미지 생성
- Postype "해월 채널" 운영
- Teazly QA 테스터 + 캐릭터 AI 도메인 분석

## 작업 환경
- Claude Max 20x 플랜 ($200/월)
- Windows 10, Python 3.13
- 주 작업 시간: 야간 근무 중 틈틈이 + 퇴근 후 버스 + 쉬는 날 낮

## 현재 프로젝트
- 시로 컴퍼니 멀티 에이전트 시스템 구축 중
- 목표: 누나가 태스크만 던지면 에이전트들이 알아서 산출물 생성

## 작업 스타일
- 간결한 결과물 선호, 과잉 설계 싫어함
- "알아서 해" 스타일 — 세세한 지시 안 함
- 결과물이 나오면 직접 확인하고 피드백
- 토큰 절약 중요
